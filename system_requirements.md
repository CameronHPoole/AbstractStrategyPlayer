## Requirements

### Section 1: Modes of operation 
1. The system shall have 3 modes of operation, autoplay, manual play, and benchmark.
   1. The system shall allow for automatic game playing between classical AI bots.
   2. The system shall allow for manual game playing between humans and bots.
   3. The system shall allow for benchmarking bot performance.

### Section 2: Game configuration
2. The system shall allow for the game board, number of players, and game dice to be configured.
   1. The game board size shall consist of 19 hexagonal tiles in a fixed or variable layout.
   2. The system shall support 2-4 players in a game where 1 of those may be a human.
   3. The game dice rolls shall be either virtual dice rolls or draws from a fixed dice decks.

### Section 3: Game display
3. The system shall display game board amd stats for the 3 game modes.
   1. In autoplay mode the game display shall show the initial and final game boards as well as the game stats for each player.
   2. In manual game mode the game display shall show each board state as moves are made, the current score, and a trade interface.
   3. In benchmark mode the game display shall show aggregated game stats for each player.
