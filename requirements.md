Requirements:

1. The system shall have 3 modes of operation, autoplay, manual play, and benchmark
2. The system shall support 2-4 players in a game where 1 of those may be a human
3. The system shall allow for automatic game playing between classical AI bots
4. The system shall allow for manual game playing between humans and bots
5. The system shall allow for benchmarking bot performance
6. Benchmarking performance shall consist of virtual dice rolls, fixed dice decks, variable boards, and fixed boards
7. The system shall display the initial and final game board in automated game mode
8. The system shall display each board state in manual game mode
